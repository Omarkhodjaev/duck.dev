## Bu darsda nimalarni o'rganasiz

- O'zgaruvchilar: `let`, `const`
- Ma'lumot turlari va operatorlar
- Funksiyalar va ularni chaqirish

> Videoni tomosha qiling, so'ng quyidagi mashqlarni o'zingiz bajarib ko'ring.

```js
const salom = (ism) => `Salom, ${ism}!`;
console.log(salom("duck.dev"));
```
